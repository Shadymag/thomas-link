Create 3 DataPages:

1.     Search.

2.     Result.

3.     Update.

 

1.     Search DataPage:

·        Type is: Submission Form.

·        Has only one virtual field, its label is: Value.

·        In “Configure Fields” screen > “Advanced” tap > “On exit” section:

§ Check “Pass field value as parameter” checkbox.

§ Write “[@Value]” in the text box.

 

·        In “Destination and Emails” screen:

§ For “Destination after record submit” choose: “Go to DataPage” option.

§ In “DataPage name” textbox write/choose: “Result” datapage.

 

2.     Result DataPage:

·        Type is: Tabular Report.

·        In “Search Type” screen:

§ Select “Filter data based on your pre-defined criteria”.

§ Check “Allow parameters in search criteria”

§ Select “Bridge and external parameters”.

 

·        In “Select Filtering Fields” screen:

§ Add only “Value” field to “Selected Fields” list.

 

·        In “Configure Fields” screen:

§ “Standard” tap:

o   For “Comparison type” choose “Equal” option.

o   Write “[@Value]” in “Value” text box.

§  “Advanced” tap > “On load” section:

o   Check “Receive value or parameter” checkbox.

o   Choose “External Parameters” option from the list.

o   Write “[@Value]” in the text box.

o   Select “Value Required” option.

 

·        In “Configure Results Page Fields” screen:

§ Add “Header”, “HTML Block”, and “Footer”.

§ In “Advanced” tap for each one of the above elements make sure to uncheck “Enable HTML editor” checkbox.

§ In “Standard” tap for “HTML Block” put the following code:

<button type="button" class="btn[@field:Review]" onclick="myTest([@field:ID])">[@field:Name]</button>

 

§ In “Standard” tap for “Footer” put the following code:

<script>

    function myTest(myID) {

 

        var element = document.createElement("iframe");

        element.style.display = "none";

        element.setAttribute('src', '[@app:SH_URL_Updating_Form]' + myID);

        document.body.appendChild(element);

 

        setTimeout(function () { document.location.reload(true); }, 2000);

       

    }

</script>

§ The above script uses application parameter “[@app:SH_URL_Updating_Form]”, its value is equal to deployment URL for “Update DataPage” which you will create in the next step. Hence you will need to finish the next step first in order to know deployment URL for “Update DataPage”, then create application parameter in that format:

§ deployment URL&ID=

§ Here is an example:

§ https://c3gfs732.caspio.com/dp.asp?AppKey=54405000fe591ede6b3840af9e9c&ID=

 

3.     Update DataPage:

·        Type is: Single Record Update Form.

·        In “Record Identification” screen:

§ Select “Find record through unique ID”.

§ For “Unique ID field” choose “ID” option.

§ Write “[@ID]” in “Parameter name” text box.

 

·        In “Select Fields” screen:

§ Select only “Review” from list.

 

·        In “Configure Fields” screen:

§ Add “HTML Block”

§ In “Advanced” tap for the above element make sure to uncheck “Enable HTML editor” checkbox.

 

§ In “Standard” tap for “HTML Block” put the following code:

<script>

    var myReview = document.getElementById('EditRecordReview');

    var intReview = parseInt(myReview.value);

 

    myReview.value = (intReview < 0 || intReview > 2) ? myReview.value = 0 : myReview.value = intReview + 1;

 

    var myForm = document.getElementById('caspioform');

    setTimeout('myForm.submit()', 1);

</script>

 

 

·        In “Destination and Emails” screen:

§ For “Destination after record update” choose: “Close window” option.

 

Finally deploy three datapages “Search”, “Result”, and “Update”. Then as an entry point use “Search” deployment URL link to start from.
